\Mbcraft\Piol

\Mbcraft\Piol__FileSystemElement

This class model a generic file system element. It is used as parent class for File and Dir classes. It has a lot of common logic for dealing with path, permissions and other file system elements metadata.

It also has methods for automatically work with file system elements storage related data.
