\Mbcraft\Piol\Utils

\Mbcraft\Piol\UtilsIniPropertiesUtils

This class contains helper methods for dealing with properties file. It actually contains some basic string and file ini operations : - read and write properties to file - read and write properties to string - add and remove properties from a file
