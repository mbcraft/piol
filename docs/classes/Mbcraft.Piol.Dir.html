\Mbcraft\Piol

\Mbcraft\PiolDir

This class enable you to easily work with directories. It contains a lot of easy to use methods for dealing with directories and its content (file listing) as well as methods for copying, moving and deleting content, as well as methods for checking parent and ancestor dirs.

It also has methods for automatically work with file system elements storage related data.
