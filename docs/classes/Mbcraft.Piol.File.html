\Mbcraft\Piol

\Mbcraft\PiolFile

This class enable you to work with files, providing a lot of easy-to-use methods for dealing with name, extension, temporary files, content and content hashing, as well as methods for moving, renaming and copying the file around.

It also has methods for automatically work with file system elements storage related data.
