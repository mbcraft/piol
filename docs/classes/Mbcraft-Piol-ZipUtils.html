ZipUtils extends PiolObject
in package
  • Application

ZipUtils.php : 17

This class provides utilities methods for extracting and compressing files using the php 'zip' extension. It has only static methods.

Actually it provides the two following operations :

  • extract an archive to a target folder
  • create an archive from a target folder, optionally putting the archived data in a custom folder.
