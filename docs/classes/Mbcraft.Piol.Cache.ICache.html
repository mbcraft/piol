has_key()

has_key(string  $key) : boolean

Returns the existence of a cache entry. An entry expires if not accessed after the *entry_expire_time*. If an entry exists it can always be accessed with get, set or delete_key. Always check an entry before accessing.

Parameters

string $key

the key of the entry to access.

Returns

boolean —

true if the entry exists and is readable, false otherwise.

Tags

api

get()

get(string  $key) : string

Returns the content of an entry as a string.

The cache entry expire time is reset for this entry. If the entry with this key does not exists this method throws an IOException.

Parameters

string $key

the string key of the entry to read

Throws

string —
