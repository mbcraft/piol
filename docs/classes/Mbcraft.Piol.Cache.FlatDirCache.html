\Mbcraft\Piol\Cache

\Mbcraft\Piol\CacheFlatDirCache

This class can be used for caching data in a flat folder fashion.

It uses as a directory for storing all the cache entries (actually 'PIOL_ROOT_PATH'/tmp/cache/ as a default folder). See the ICache interface definition for more details.
